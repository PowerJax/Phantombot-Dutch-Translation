Commands

(sender) Persoon die de command heeft getriggerd.
(@sender) Persoon die de command heeft getriggerd en voegt een @ symbool toe aan zijn naam.
(baresender) Persoon die de command heeft getriggerd in kleine letters.
(touser) Wordt vervangen met het argument dat na de command gezegt word. Als er niks gezegt wordt na de command is het de naam van de persoon die de command heeft getriggerd.
(#) willekeurig getal
(1), (2), (3) etc command argumenten.
(age) Verteld je hoe oud je Twitch account is.
(random) willekeurig persoon in de chat.
(pointname) Huidige punten naam.
(uptime) Huidige tijd dat de stream online is.
(game) Huidig spel dat is ingesteld op Twitch.
(status) Huidige titel ingesteld op Twitch.
(price) Huidige bevel prijs.
(pay) betaling van de command.
(onlineonly) Zorgt ervoor dat de command alleen werkt als de stream online is.
(offlineonly) zorgt ervoor dat de command alleen werkt als de stream offonline is.
(viewers) Het huidige aantal kijkers.
(count) Hoevaak de command gebruikt is.
(echo) zegt een bericht als de bot.
(1=) Argument 1. Add a the tag name of your choice after the = for if the argument is empty.
(countdown=) Voeg een datum toe waar je naar wilt aftellen. Voorbeeld (countdown=July 2 2016 7:00).
(downtime) Tijd sinds de laatste stream.
(readfile filename.txt) Leest wat is in het bestand. Het bestand moet in de bots addon folder zitten.
(customapijson URL key | {literal string} ...) Call a remote URL that returns JSON. key or object.key or object.object.key to access JSON data.
(command [command_name]) roep een command op. Must be first item in custom command. Do not include ! before command_name
(customapi [requesturl]) Call a remote URL
